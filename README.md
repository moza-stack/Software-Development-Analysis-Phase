| #  | User Story                                                                                                      | Acceptance Criteria                                                                                                        | Story Points / Weight |
| -- | --------------------------------------------------------------------------------------------------------------- | -------------------------------------------------------------------------------------------------------------------------- | --------------------- |
| 1  | As a patient, I want to register in the hospital system so that I can book appointments.                        | Patient registration form saved with unique ID; validation of required fields; confirmation notification.                  | 5                     |
| 2  | As a doctor, I want to view my patient list so that I can manage my appointments efficiently.                   | Doctor can see all assigned patients; search/filter options available; list updated in real-time.                          | 3                     |
| 3  | As a patient, I want to book an appointment online so that I can avoid waiting in the hospital.                 | Patient can select doctor, date, and time; appointment confirmation sent via email/SMS; prevents double booking.           | 5                     |
| 4  | As a receptionist, I want to view all scheduled appointments so that I can manage the daily schedule.           | Daily/weekly/monthly view of appointments; ability to update or cancel; status tracking of each appointment.               | 3                     |
| 5  | As a doctor, I want to update patient medical records after consultation so that all data is current.           | Patient history updated; notes, prescriptions, and vitals saved; data visible to authorized staff only.                    | 8                     |
| 6  | As a lab technician, I want to receive test requests and update results so that doctors can access them.        | Lab requests generated from doctor orders; results uploaded securely; notification sent to doctor.                         | 5                     |
| 7  | As a patient, I want to view my medical records online so that I can track my health history.                   | Secure login; records include visits, prescriptions, lab results; download option available.                               | 5                     |
| 8  | As a billing officer, I want to generate patient bills automatically so that billing is accurate and fast.      | Bills generated with services, tests, and medication costs; payment status tracked; invoice PDF downloadable.              | 8                     |
| 9  | As a patient, I want to make payments online so that I can settle bills conveniently.                           | Payment gateway integrated; success/failure notification; transaction recorded in system.                                  | 5                     |
| 10 | As a pharmacy staff, I want to manage medicine inventory so that stock is available when needed.                | Inventory updated with purchases, prescriptions; low-stock alerts; reports generated for management.                       | 5                     |
| 11 | As a hospital admin, I want to generate reports on daily operations so that I can monitor performance.          | Reports on patient visits, appointments, revenue, lab tests; exportable to PDF/Excel.                                      | 8                     |
| 12 | As a patient, I want to receive reminders for upcoming appointments so that I don’t miss them.                  | Notifications sent via email/SMS 24 hours before; option to reschedule; confirmation tracked.                              | 3                     |
| 13 | As a doctor, I want to prescribe medications digitally so that patients can access them at the pharmacy easily. | Prescription linked to patient record; pharmacy notified; patient can view online.                                         | 5                     |
| 14 | As a system admin, I want to manage user roles and permissions so that data security is maintained.             | Admin can assign roles (doctor, nurse, receptionist, lab, pharmacy); permissions enforced per role; audit logs maintained. | 8                     |
| 15 | As a patient, I want to provide feedback on services so that the hospital can improve care.                     | Feedback form available; stored securely; summary report viewable by admin.                                                | 3                     |
